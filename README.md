# Chenning Wu Personal Homepage

Static personal homepage for GitHub Pages.

## Files

- `index.html` - homepage content and structure
- `styles.css` - responsive visual design
- `assets/research-hero.png` - generated hero image
- `assets/chenning-wu-resume.pdf` - downloadable CV

## Publish on GitHub Pages

1. Create a public repository named `your-github-username.github.io`.
2. Upload these files to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`, choose `main`, and use `/root`.
5. Your site will be available at `https://your-github-username.github.io/`.
